Product Management System

Full-Stack Enterprise Web Application

A scalable and secure enterprise-level application developed using ASP.NET Core Web API, React.js, SQL Server, and Entity Framework Core. The system focuses on secure authentication, product management, category handling, dashboard analytics, and role-based access control.

Authentication & Login Module

Implemented a secure JWT-based authentication system with protected API access and role-based authorization. Users can securely log in using email and password credentials while maintaining protected sessions.

Dashboard Management

Designed an enterprise-style admin dashboard displaying real-time statistics for products, categories, users, and franchises. The dashboard provides centralized monitoring and operational insights for administrators.

Product Management Module

Developed complete CRUD operations for product management including product creation, editing, deletion, category mapping, filtering, pricing management, and manufacturer details.

The system integrates RESTful APIs with SQL Server using Entity Framework Core for optimized and scalable data operations.

Category Management Module

Implemented category and sub-category management with secure validation workflows. Categories connected to products cannot be deleted, ensuring relational integrity and safe business operations.

This module demonstrates practical enterprise business-rule validation and structured inventory organization.

Role-Based Authorization

Implemented secure role-based authorization for Admin, Franchise, and User roles. Protected APIs and restricted UI functionalities ensure secure enterprise-level access management.

Admin

Full access to products, users, categories, dashboards, and franchise operations.

Franchise

Limited operational access for franchise-level management.

User

Basic authenticated user access with protected operations.

Backend Architecture

Built using ASP.NET Core Web API with Entity Framework Core and SQL Server. Applied clean architecture principles including DTOs, service layers, separation of concerns, and scalable REST API design.
